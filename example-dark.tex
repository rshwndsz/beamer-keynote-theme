\begin{frame}{Speeding up Inference}{KV Cache}
  The standard attention computation:
  \[
    \text{Attention}(Q, K, V) = \text{softmax}\!\left(\frac{QK^T}{\sqrt{d_k}}\right) V
  \]

  \vskip1em

  Key insight: during autoregressive generation, we only need the
  query vector for the \emph{new} token. The $K$ and $V$ vectors for
  all previous tokens can be cached and reused.
\end{frame}

\begin{frame}{Speeding up Inference}{Speculative Decoding}
  \begin{itemize}
    \item Use a smaller draft model to generate $k$ candidate tokens
    \item Verify all $k$ tokens in a single forward pass of the large model
    \item Accept prefix of tokens that match; reject and resample the rest
    \item Expected speedup: $\sim 2\text{--}3\times$ with no quality loss
  \end{itemize}
\end{frame}
